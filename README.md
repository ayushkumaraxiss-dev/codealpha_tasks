# 🎮 Hangman Game in Python

This is a simple Hangman game made using Python. The player has to guess the hidden word one letter at a time before running out of lives.

# Features

* Random word selection
* 6 lives system
* Visual Hangman stages
* Win and lose conditions
* Easy, Medium, and Difficult words
* User-friendly console interface with emojis

# How to Play

1. Run the Python file.
2. A random word will be selected.
3. Guess one letter at a time.
4. Correct letters are revealed in the word.
5. Wrong guesses reduce your lives.
6. Guess the complete word before your lives reach zero.

# Files Used

* `main.py` - Main game logic
* `word_file.py` - Contains the word list
* `Hangman_stages.py` - Stores Hangman ASCII art stages

# Technologies

* Python
* Random Module
